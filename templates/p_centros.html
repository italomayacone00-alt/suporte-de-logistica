📖 Entenda o Problema dos p-Centros

🎯 O Que é o Problema dos p-Centros?

analytics

O problema dos p-Centros é um modelo de otimização min-max que seleciona exatamente p instalações (centros) para minimizar a maior distância ou maior custo entre qualquer cliente e o centro que o atende.


Diferença Chave: Ao contrário das p-Medianas (minimiza soma total), os p-Centros focam no pior caso - garantindo que nenhum cliente fique excessivamente longe de um centro.


Contexto Histórico: Introduzido por Hakimi (1964) em seu trabalho seminal sobre "Optimum Locations of Switching Centers and the Absolute Centers and Medians of a Graph", revolucionou a localização de instalações com aplicações críticas onde o tempo máximo de resposta é vital.

🧮 Formulação Matemática Detalhada

functions
Função Objetivo
Minimização
Minimizar
Z ≥ dij × Xij ∀ i∈I, j∈J

Onde:

  • Z = Variável auxiliar que representa a máxima distância/custo de atendimento
  • dij = Distância/custo entre instalação i e cliente j
  • Xij = Variável binária (1 se cliente j é atendido pelo CD i, 0 caso contrário)
  • Yi = Variável binária (1 se instalação i está aberta, 0 caso contrário)
  • ∀ i∈I, j∈J = Para todas as instalações i e clientes j

Explicação: O modelo minimiza Z, que é forçado a ser maior ou igual a todas as distâncias de atendimento. Assim, Z representa o pior caso (maior distância) que queremos minimizar.

rule
Restrições Principais
Obrigatórias
Σ i∈I Yi = p
Número de Instalações

Exatamente p instalações devem ser abertas/ativadas

Σ i∈I Xij = 1 ∀ j∈J
Atendimento Completo

Cada cliente deve ser atendido por exatamente uma instalação

Xij ≤ Yi ∀ i∈I, j∈J
Vínculo Lógico

Um cliente só pode ser atendido por uma instalação que esteja aberta

Xij, Yi ∈ {0,1} ∀ i∈I, j∈J
Natureza das Variáveis

Variáveis binárias (Programação Linear Inteira)
