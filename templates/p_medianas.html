Sobre as p-Medianas

Este modelo seleciona exatamente p localizações para minimizar a soma ponderada de distâncias ou custos. Cada cliente é atendido pelo CD mais próximo, considerando a procura. Ideal para otimização global de custos.

📖 Entenda o Problema das p-Medianas

🎯 O Que é o Problema das p-Medianas?

analytics

O problema das p-Medianas é um modelo clássico de otimização que busca selecionar exatamente p instalações (medianas) para atender um conjunto de clientes, minimizando a soma total das distâncias ponderadas.


Contexto Histórico: Introduzido por Hakimi (1964), é considerado um dos problemas mais importantes na localização de instalações com aplicações práticas em logística, telecomunicações, serviços de emergência e planejamento urbano.

functions
Função Objetivo

Minimizar
Σ
i∈I j∈J
dij × qj × Xij

Onde:

  • dij = Distância/custo entre CD i e cliente j
  • qj = Demanda do cliente j
  • Xij = Variável binária (1 se cliente j é atendido pelo CD i, 0 caso contrário)
rule
Restrições Principais
Obrigatórias
Σ i∈I Yi = p
Número de Instalações

Exatamente p CDs devem ser abertos/ativados

Σ i∈I Xij = 1 ∀ j∈J
Atendimento Completo

Cada cliente deve ser atendido por exatamente um CD

Xij ≤ Yi ∀ i∈I, j∈J
Vínculo Lógico

Um cliente só pode ser atendido por um CD que esteja aberto

Xij, Yi ∈ {0,1} ∀ i∈I, j∈J
Natureza das Variáveis

Variáveis binárias (Programação Linear Inteira)

psychology

Conceito

hub

Cada CD selecionado atende como "hub" central para seu cluster de clientes

Objetiva reduzir a soma TOTAL das distâncias, não individualmente

Tamanho do Espaço de Busca
C(m,p) = m! / [p! × (m-p)!]

Onde m = CDs candidatos, p = CDs a selecionar
