Sobre a Máxima Cobertura

Este modelo seleciona exatamente p instalações para maximizar a cobertura ponderada dentro de um raio predefinido. Ideal para serviços onde o alcance geográfico é o fator principal.

Entenda o Problema da Máxima Cobertura

O Que é o Problema da Máxima Cobertura?

radar

O problema da Máxima Cobertura é um modelo de otimização que seleciona exatamente p instalações para maximizar a cobertura ponderada dentro de um raio predefinido, onde cada ponto tem um peso específico (população, demanda, etc.).


Diferença Chave: Ao contrário dos p-Centros (minimiza maior distância) e p-Medianas (minimiza soma total), a Máxima Cobertura foca em maximizar o número de pontos atendidos dentro de um limite de distância.


Contexto Histórico: Introduzido por Church e ReVelle (1974) em seu trabalho seminal sobre "The Maximal Covering Location Problem", revolucionou a localização de serviços com aplicações críticas onde o alcance geográfico é o fator principal.

Formulação Matemática

functions
Função Objetivo
Maximização
Maximizar
Σ
i ∈ I
wi × yi

Onde:

  • wi = Peso do ponto i (população, demanda)
  • yi = 1 se ponto i é coberto, 0 caso contrário
  • I = Conjunto de todos os pontos de cobertura

Explicação: O modelo maximiza a soma dos pesos dos pontos cobertos, onde um ponto é considerado coberto se está dentro do raio de pelo menos uma instalação selecionada.

rule
Restrições Principais
Obrigatórias
Σ j∈J xj = p
Número de Instalações

Exatamente p instalações devem ser selecionadas

Σ j∈Ni xj ≥ yi ∀ i∈I
Garantia de Cobertura

Ponto i só é coberto se pelo menos uma instalação j dentro do raio for selecionada

xj, yi ∈ {0,1} ∀ j∈J, i∈I
Variáveis Binárias

Instalações e pontos só podem ser selecionados (1) ou não (0)
